Réalisations

  • Création d'un jeu avec la méthode agile - BUT2

    Java, Méthode Agile
    -
    Avec mon équipe, création d'un jeu de hasard se déroulant sur un plateau de Monopoly.
    Le principe est de vaincre des monstres en améliorant son équipement et ses statistiques afin de faire pencher la chance en sa faveur.
    Nous avions 3 jours pour créer ce jeu en utilisant la méthode agile.

  • Exploitation d'une Base de Données - BUT1

    PostgreSQL
    -
    Avec mon binôme, nous avons dû exploiter une base de données sur tous les Jeux Olympiques ayant eu lieu jusqu'à présent.
    Nous avons créé des requêtes pour récupérer des données de la base, et pour cela, nous l'avons ventilée afin d'optimiser son traitement.

  • Création d'un jeu Ludo-pédagogique - BUT1

    Ijava
    -
    Création d'un jeu nommé UntitledAdventure en équipe de deux.
    Le jeu consiste à compléter des mondes eux mêmes constitués d'un certains nombre d'ennemis à vaincre, et pour leur infliger des dégats,
    il suffit de répondre correctement à des questions sur diverses matières scolaires.
    Plus on avance dans le jeu, plus les monstres ont de vies et plus ils infligent de dégâts.

  • Création d'un site web - BUT1

    HTML, CSS
    -
    Par groupe de 3, création d'un site WEB fictif afin de faciliter la mobilité des employés d'une entreprise.
    Nous avons choisi de réaliser notre site WEB pour l'entreprise Ubisoft.

  • Jeu chasse au monstre - BUT3

    Java, modèle MVC
    -
    Nous avons par groupe de 4, créée un jeu vidéo en java se jouant à 2 joueurs. Il y a deux équipes, celle du monstre qui doit s'échapper d'un labyrinthe lui étant inconnu. L'autre équipe est celle du chasseur, qui lui doit débusquer le monstre en ayant aucune connaissance du labyrinthe.

  • API pizzaland - BUT4

    Java, API REST, SQL
    -
    A l'aide d'un autre camarade de classe, nous avons mis au point une api rest pour une pizzeria fictive permettant de créer des pizzas pour la carte, ajouter des ingrédients, créer et gérér des commandes...

  • Salamander story - BUT4

    JavaScript, modèle MVC, HTML, CSS
    -
    En binome, nous avons développé un jeu du style Shoot'em up jouable en multijoueur. Celui-ci possède une bonne diversité d'ennemies différents, de projectiles différents et autres mécaniques de jeu. Absolument tout le projet a été fait par nous, cela va jusqu'aux assets du jeu (graphismes, sons).
